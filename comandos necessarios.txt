coisas para fazer uma unica vez no computador:
 
// instalar o typescript para o computador inteiro

npm install -g typescript
 
//verificar se ta ok o Typescript

tsc -v
 
//instalar o ts-node (carinha que roda o projeto)

npm install -g ts-node
 
//verificar a versão do ts-node:

ts-node -v

//Instalar do readline-sync

npm i --save-dev @types/readline-sync

 
iniciar um novo projeto com TS

// dentro da pasta do projeto

npm init -y
 
tsc --init
